# ESLint Plugin yamatomo

## Installation

```bash
$ npm i --save-dev @yamatomo/eslint-plugin
```

## Usage

```json
{
  "extends": ["plugin:@yamatomo/base"]
}
```

## rules

| ------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------- |
| [`@yamatomo/no-blacklist-imports`](./docs/rules/no-blacklist-imports.md)                                     | Prevent disallowed module imports                                                                                       |
